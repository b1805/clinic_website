🦜 Finder Information

Search, merge, and download ecological & biological datasets in one place.

What is the Finder?

The Finder is a data retrieval tool that integrates multiple ecological and biological datasets into a single interface. You can search by species name, category, or ID, and download the matching data as an Excel file.

How It Works

  • Search by Species Name: Enter a full or partial species name, or a broader category like a genus, family, or "ALL DATA".
  • Search by ID: Toggle “Search by ID” to enter known Binary_ID or Species_ID values.
  • Trait Selection: Required traits are always included. You can also choose optional traits from the list.
  • Merging Type:
    • Merged Columns – combines equivalent traits from different datasets into single columns.
    • Unmerged Columns – keeps traits from each dataset separate.
  • Advanced Mode: Allows adjusting fuzzy matching threshold for partial matches.
  • Test Mode: Compare generated results against a reference Excel file to get similarity and quality scores.

Outputs

The Finder returns results in Excel format. Depending on your selection, the file may contain:

  • Required metadata: Source, species, common_name, family, order, genus
  • Optional traits selected by the user
  • Resolution errors (if any) in a separate downloadable file

Tips for Best Results

  • Use scientific species names for the most accurate matches.
  • When using fuzzy matching, lower thresholds will return more potential matches but may include false positives.
  • If searching large datasets, try narrowing down traits to reduce file size.

Example Searches

  • Species Name: Corvus corax → returns data for the common raven.
  • Category: Passeriformes → returns all passerine birds.
  • ID Search: Binary_ID = 150914.0saxspa1
